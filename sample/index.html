Task Tracker

Welcome to The Task Tracker. This is a tool developed by Matt Bedynek to demonstrate the use of PHP and MySQL integration.

The summary (main) page of the tracker can be accessed at http://no-carrier.org/tasker/list.php

Future versions will add support for...

  • E-mail and text notifications.
  • Add Timezone support.
